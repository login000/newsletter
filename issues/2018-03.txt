another amazing month around the town as we have new software, new
activities and entirely new digital experiences!

~ HOLODECK ~
wow! this is amazing! tilde.town hero ~ne1 has created holodeck, a shared
experience / storytelling tool where you can navigate a user-created
world and add to the shared text. check out the project page here:
https://tilde.town/~ne1/code.html

to check out this amazing tool just run ~ne1/bin/holodeck.

~ SOLITAIRE ~
we now have a solitaire game on tilde.town! wow! relive wasted hours when
you should have been filling in spreadsheets in windows 95 by running
usolitaire at the command prompt. cards bouncing around the screen when
you win so far unconfirmed.

~ BOOK CLUB ~
tilde.town mythical figure ~wuz has started a book club! the current book
is 'What is Property?' and you can find a link to it on the book club's
page: http://tilde.town/~wuz/tilde_book_club

time to open-palm slam a paperback into your bookshelf and slip on your
reading goggles, it's literature time book fans!
